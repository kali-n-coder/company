# 情報セキュリティルール

## 目的

会社情報、顧客情報、個人情報、アカウントを安全に守るためのルールです。

## アカウント管理

- 会社アカウントを他人に貸さない。
- パスワードをメモ帳やチャットにそのまま書かない。
- 退職者や利用しないアカウントは速やかに停止する。
- 可能なサービスでは二段階認証を利用する。

## パスワード

- 推測されにくいパスワードを使う。
- 他サービスと同じパスワードを使い回さない。
- 漏えいの疑いがある場合はすぐに変更する。

## データの持ち出し

- 顧客情報や会社資料を、許可なく外部へ送らない。
- 私物PCや私物USBへの保存は原則禁止とする。
- 業務上必要な場合は、事前に管理者の許可を得る。

## 端末管理

- PCやスマホには画面ロックを設定する。
- 紛失や盗難があった場合は、すぐに会社へ連絡する。
- ウイルス対策やOSアップデートをできるだけ最新に保つ。

## インシデント対応

以下が起きた場合は、すぐに管理者へ報告します。

- アカウントの不正利用の疑い
- 端末の紛失
- 誤送信
- 顧客情報の漏えい疑い
- 不審なメールやリンクを開いた場合
