# 会社ルール・書類テンプレート集

会社で決めておくと運用が楽になる書類テンプレートです。
このフォルダのMarkdownは、社内用のたたき台として自由に書き換えて使う想定です。

正式な就業規則、賃金規程、個人情報関連書類などは、会社規模や業種によって必要な内容が変わります。実際に提出・配布する前に、社労士、税理士、弁護士などの専門家確認を入れると安全です。

## 使い方

1. 自社に合うものからコピーして編集する。
2. 「未定」「要確認」と書いてある部分を会社の実態に合わせて埋める。
3. 従業員へ共有する前に、責任者が内容を確認する。
4. 重要な規程は専門家に確認してもらう。
